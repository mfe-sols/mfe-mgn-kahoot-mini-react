# MFE MGN Kahoot Mini React

Mini quiz game 15 questions built from the React single-spa template.

## Run

- `pnpm -C apps/mfe-mgn-kahoot-mini-react start`
- Open standalone at `http://localhost:19114`
- Or run root-config and open `http://localhost:9000/mgn-kahoot-mini`

## Purpose

- Demo a self-contained game module inside the shell
- Validate that a new React MFE can be mounted from `root-config`
- Keep the same MVP folder structure as the shared template
